Installation instructions

Download

Download the latest version of the standalone application DownZemAll (DownRightNow) for your platform/OS here:

https://setvisible.github.io/DownZemAll/

(or here: https://setvisible.github.io/DownZemAll/category/download.html)

Install

For Windows users:

To install the application:

  • Extract the downloaded ZIP file in a local directory.
  • Double-click on "install.bat".
  • Wait for the script to display the successful message.

To uninstall the application:

  • Double-click on "uninstall.bat".
  • Wait for the script to display the successful message.
  • Remove the directory.

Note:

These scripts do not need administrator permission.
